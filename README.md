# 🍽️ Savory Bites — Premium Restaurant Management System

A modern full-stack restaurant management web application built using the MERN stack. The project provides a premium restaurant-style UI with complete CRUD functionality for menu management and consumer registration.

---

# ✨ Features

## 🎨 Frontend Features

* Premium luxury restaurant UI
* Dark Charcoal + Gold + Cream color theme
* Fully responsive design
* Modern hover animations and transitions
* Dynamic menu cards with food images
* Category-based filtering
* Search functionality
* Price filtering
* Dashboard analytics cards
* Toast notifications
* Consumer registration form

---

## ⚙️ Backend Features

### Menu CRUD

* Add menu items
* Fetch menu items
* Update menu items
* Delete menu items

### Consumer CRUD

* Register consumers
* Fetch registered consumers
* Update consumer data
* Delete consumers

---

# 🛠️ Tech Stack

## Frontend

* React.js

* React Router DOM

## Backend

* Node.js
* Express.js
* MongoDB
